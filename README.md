# Portfólio — Tommy.dev

Portfólio pessoal de desenvolvimento web. Site estático em HTML, CSS e JavaScript, com seletor de idioma PT/EN.

**Site:** https://tommywebdesign.netlify.app

## Estrutura

- `index.html` — página principal
- `style.css` — estilos
- `img/` — imagens do portfólio
- `car/`, `WeatherApp/`, `YourLife/` — projetos de demonstração

## Deploy

O site é publicado automaticamente no Netlify a cada `git push` para o branch `main`.
